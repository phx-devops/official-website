This 3-day training program provides foundational knowledge in DevOps, infrastructure automation, and pipeline security. Each day includes hands-on exercises and practical examples.

## Day 1: DevOps Fundamentals

**Duration:** 8 hours (9 AM - 5 PM)

### Morning Session: Introduction to DevOps
- What is DevOps? Culture, practices, and principles
- DevOps lifecycle: Plan, Code, Build, Test, Release, Deploy, Operate, Monitor
- Benefits of DevOps: Faster delivery, improved quality, better collaboration
- Hands-on: Setting up a basic development environment

### Afternoon Session: Version Control and Collaboration
- Git basics: Commits, branches, merges, and pull requests
- Collaborative workflows: GitFlow and trunk-based development
- Code review best practices
- Hands-on: Creating a Git repository and collaborating on code

## Day 2: Infrastructure Automation

**Duration:** 8 hours (9 AM - 5 PM)

### Morning Session: Infrastructure as Code (IaC)
- Introduction to IaC concepts and benefits
- Overview of popular IaC tools: Terraform, Ansible, CloudFormation
- Writing declarative infrastructure code
- Hands-on: Creating and deploying infrastructure with Terraform

### Afternoon Session: Configuration Management
- Configuration management principles
- Automating server configuration and application deployment
- Containerization basics with Docker
- Hands-on: Building and deploying containerized applications

## Day 3: Pipeline Security

**Duration:** 8 hours (9 AM - 5 PM)

### Morning Session: CI/CD Pipeline Basics
- Understanding continuous integration and continuous deployment
- Building secure pipelines with tools like Jenkins, GitLab CI, or GitHub Actions
- Automated testing strategies
- Hands-on: Creating a basic CI/CD pipeline

### Afternoon Session: Security in DevOps
- DevSecOps principles: Shifting security left
- Common security vulnerabilities and how to prevent them
- Implementing security scanning in pipelines (SAST, DAST, SCA)
- Hands-on: Adding security checks to a CI/CD pipeline

## Prerequisites
- Basic understanding of software development
- Familiarity with command-line interfaces
- Laptop with internet access for hands-on exercises

## What You'll Learn
By the end of this training, participants will be able to:
- Understand core DevOps principles and practices
- Automate infrastructure provisioning and configuration
- Build and secure CI/CD pipelines
- Apply DevOps methodologies to improve development workflows
